# To-Do List App

This is a simple To-Do List App built using HTML, CSS, and JavaScript.

## Features

1. Add tasks with a title and optional description.
2. Mark tasks as completed.
3. Edit existing tasks.
4. Delete tasks.
5. View a list of tasks.
